博客命令介绍

hexo n “新建文章名称” == hexo new “新建文章” #新建文章.

hexo g == hexo generate #生成博客文件.

hexo s == hexo server #启动博客服务.

hexo d == hexo deploy #部署博客.

hexo clean #清除缓存.
